e shift (Python)


    text = input()
    shift = int(input())

    encrypted = ""

    for c in text:
        if c.isalpha():
            encrypted += chr((ord(c.lower()) - 97 + shift) % 26 + 97)
        else:
            encrypted += c

    print(encrypted)

Most Frequent IP (Python)

    ips = f.read().split()
